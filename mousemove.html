pan.js adds native-like panning functionality using JavaScript and CSS3. Optimized for iPad, iPhone and desktop browsers.

Move the mountains using you finger or mouse. You can also pan using click/drag as seen here.

No dependencies, 1.5k gzipped.

Tested in IE7+, FF3+, Opera 10+, Chrome, Safari, iPad 1 and iPhone 4. MIT License.
